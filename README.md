# studying-reactNode
react.js, node.js를 이용한 간단한 웹사이트 만들기

### 선행 작업
1. node 설치
2. mysql 설치
3. 패키지 설치
```
npm install express
npm install mysql
npm install -g nodemon
npm install react-router-dom
npm install axios
```

### 서버 실행 시 
1. react 폴더에서 ```npm run build``` 
2. server.js 폴더에서 ```nodemon server.js```
